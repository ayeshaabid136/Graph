# Graph

🔹 Overview

This project implements a Graph using Adjacency List in C++.

It supports weighted, directed and undirected graphs and includes multiple graph operations.

---

🔹 Features

🧱 Basic Operations

- Add Vertex
- Add Edge (Directed / Undirected with weight)
- Remove Vertex
- Remove Edge
- Display Graph

---

🔍 Traversal

- BFS (Breadth First Search) using queue

---

📊 Degree Functions

- In-Degree
- Out-Degree
- Sum of Degrees

---

🛣️ Path Finding

- Find path between two vertices using BFS
- Uses "parent" pointer

---

🔗 Graph Checks

- Check if graph is Connected
- Check if graph is Cyclic

---

🔎 Utility Functions

- Search Vertex
- Check Adjacency
- Find Neighbours

---

🌳 MST Algorithms

- Kruskal Algorithm
- Prim Algorithm

---

🔹 Concepts Used

- Linked List (Multi-linked structure)
- Queue (BFS)
- Pointers
- Dynamic Memory Allocation

---

🔹 Notes

- Graph is fully dynamic
- Uses simple logic (no STL graph library)
- Fixed size arrays used in MST (size = 20)

---

🔹 Conclusion

This project helps in understanding core graph concepts like:

- Traversal (BFS)
- Graph properties
- Minimum Spanning Tree algorithms
